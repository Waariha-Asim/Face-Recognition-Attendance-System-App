# 🎯 Face Recognition Attendance System

> An AI-powered Android application that automates attendance management using **Face Recognition**, **TensorFlow Lite (FaceNet)**, **Google ML Kit**, and **Firebase** for secure authentication and real-time attendance tracking.

---

# 📌 Overview

The **Face Recognition Attendance System** is an intelligent Android application developed to modernize the traditional attendance process through AI-powered facial recognition.

The application enables students to register and verify their identity using facial recognition while allowing teachers to efficiently manage and monitor attendance records. By integrating **TensorFlow Lite**, **Google ML Kit**, **CameraX**, and **Firebase**, the system delivers a secure, reliable, and user-friendly attendance solution.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Waariha-Asim/Face-Recognition-Attendance-System-App.git
```

## Setup

1. Open the project in Android Studio.
2. Sync Gradle dependencies.
3. Configure Firebase.
4. Add your `google-services.json` file to the `app/` directory.
5. Build and run the application.
